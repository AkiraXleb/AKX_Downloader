# 📦 Installation

## Clone repository

```bash
git clone https://github.com/YOUR_USERNAME/AKX-Downloader.git
```

---

## Install dependencies

```bash
pip install -r requirements.txt
```

---

# 🚀 Run

```bash
python AKX_Downloader2.0.py
```

---

# 🏗️ Build EXE

```bash
pyinstaller --onefile --windowed --icon=icon.ico AKX_Downloader2.0.py
```
